# SPL Draft Game

SPL Draft is a football squad-building and season-simulation game based on Saudi Pro League history.

Players draft an eleven-player squad using cards from 26 seasons, covering 2000/01 through 2025/26, and then simulate a complete league season.

## Production Version

Current stable release:

`v7.0-production-stable`

## Features

- 11,282 historical player cards
- 26 Saudi Pro League seasons
- Standard Draft mode
- Daily Challenge mode
- Full-season simulation
- Player and season awards
- Standard and Daily leaderboards
- Secure server-side gameplay and scoring
- Draft and season recovery after refresh
- Cloudflare Turnstile protection
- Supabase backend and authentication
